Purpose

The assignments purpose is to learn how to use HTML using some Linux commands in git. Below is a list of some commands included git, VI, and linux.

Some Examples of Linux Commands

			ls = directory listing
			rm -rf dir = force remove directory dir
			cd = change directory
			cp –r dir1 dir2 = copy dir1 to dir2; create dir2 if it doesn’t exist
			tail-f file = output the connects of file as it grows, starting with last 10 lines
			pwd = present working directory
			mkdir = make directory
			rm = delete

List of VI Command Examples

	                vi filename = enter file and view code
	                i = insert text
  			:wq = write and quit
	                :w = write
	                :q = quit

List of Git Commands

	                git status = show file in directory
	                git add = add file to commit
			git commit -m = commit the file and add description
	                git merge = merge branch with main
